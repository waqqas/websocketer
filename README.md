# Websocketer

A header-only websocket client library built on boost::beast library

Building library
---

- `mkdir build`
- `cd build`
- `cmake -DCMAKE_INSTALL_PREFIX:PATH=`pwd`/usr ..`
- `cmake --build . --config Release --target install -- -j $(nproc)`

Running tests
---

- `./tests/tests`
